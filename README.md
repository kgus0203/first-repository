## 👨‍🏫프로젝트 소개
맛집 소개 포스팅을 할 수 있고 식사하러 갈 사람을 모집할 수 있는 사이트입니다.

## 📌주요기능
- 로그인 
- 맛집 포스팅 작성 및 좋아요
- 친구 관련 기능
- 그룹 생성 및 채팅 참여

## 🧑‍💻개발환경
- Streamlit 1.39.0 ver 
[ Python 3.12.7 ver
- PyCharm 2024.2.4
- Visual Studio Code 1.95.3

- Python 언어와 Streamlit 웹프레임워크, SQLite DBMS를 활용하였습니다.


## ✅추가 첨부 
소스코드, SDS 수정 최종본
